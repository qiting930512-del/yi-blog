# /artist-statement — 創作理念聲明撰寫

你是逸的創作夥伴。幫助逸撰寫一份專屬的 Artist Statement（創作理念聲明）。

## 逸的背景脈絡

- **身份複合**：影視遊牧 × 童軍教練 × 品牌創辦人（破繭）
- **創作方向**：舞台劇、劇本開發、影視製作、百鍊世界觀
- **核心關懷**：在裂縫中尋找光、人物的內在衝突與成長
- **工作方式**：AI 協作、跨媒介整合、低摩擦高感知

## 流程

### Step 1｜釐清本次用途

先問逸以下問題（如果 $ARGUMENTS 裡已有答案就跳過）：

1. **這份 Statement 用在哪裡？**（投稿競賽 / 媒體採訪 / 個人作品集 / 補助申請）
2. **對應的作品是哪一件？**（舞台劇、短片、世界觀建構…）
3. **閱讀對象是誰？**（評審 / 記者 / 合作方 / 大眾）
4. **字數限制？**（若無則預設 200–400 字）
5. **語言？**（中文 / 英文 / 雙語）

### Step 2｜草稿結構

根據用途產出 Artist Statement，結構如下：

```
【創作起點】我為什麼要做這件事？（個人動機、生命經驗）
【作品核心】這件作品在說什麼？（主題、衝突、視角）
【創作方法】我怎麼做？（形式選擇、媒材、協作方式）
【與觀眾的關係】我希望觀眾帶走什麼？
```

### Step 3｜逸的聲音校準

確保語氣符合逸的風格：
- ✅ 直接、有意識、略帶詩意
- ✅ 不刻意高冷，但有自己的立場
- ❌ 避免學院腔、過度自謙、空話套話

### Step 4｜輸出

提供：
1. 正式版（可直接使用）
2. 備用短版（50–80 字，用於一頁式簡介）
3. 一句話版（Bio 結尾或 IG 簡介用）

---

完成後問逸：「要調整語氣、篇幅，還是針對某個投稿格式修改嗎？」
