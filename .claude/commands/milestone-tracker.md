## 流程

### Step 1｜從 Notion 取得里程碑

使用 Notion MCP 工具，查詢統一任務中心（collection://368f02a3-07a9-8017-a75c-000b1bab2750）中：
- 狀態為「下一步」、「進行中」、「等待中」的任務
- 有截止日的任務（依截止日升冪排序）

若無法連接 Notion，請逸手動告知各專案的最新進度。

### Step 2｜里程碑倒數看板

輸出格式如下：

```
📅 今天：[日期]

⚠️ 緊急（7 天內）
├─ [專案名] ── [里程碑] ── 距今 X 天

🔶 本月（30 天內）
├─ [專案名] ── [里程碑] ── 距今 X 天

📌 待確認截止日
├─ [專案名] ── [目前狀態] ── 下一步？
```

### Step 3｜衝突偵測

檢查是否有里程碑在同一週內重疊，若有則提示：
> ⚡ 注意：[專案A] 和 [專案B] 截止日相近，建議提前規劃緩衝時間。

### Step 4｜本週建議行動

根據里程碑倒數，輸出「本週最重要的 3 件事」：

```
本週 Top 3：
1. [任務] ── [原因]
2. [任務] ── [原因]
3. [任務] ── [原因]
```

---

結束後問逸：「要更新哪個專案的截止日，或是加入新里程碑嗎？」
